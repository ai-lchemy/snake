# Terminal Snake

Run with Python 3 in a terminal:

```sh
python3 snake.py
```

Use the arrow keys or WASD to move. Press P (or Escape) for the pause menu and Q to quit.
The game accelerates as food is eaten and stores the three best scores in
`~/.snake_highscores.json`. Use `--scores PATH` to choose another score file.

Run the tests with `python3 -m unittest -v`.
