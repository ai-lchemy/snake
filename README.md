# Terminal Snake

A dependency-free Snake game for Python 3.10 or newer.

Run it in an interactive terminal:

```sh
python3 snake.py
```

Use the arrow keys to steer, `P` to open the pause menu, and `Q` to quit. The pause menu can resume or restart the game. The snake speeds up during play and as food is eaten, and the top three scores are saved in `~/.snake_highscores.json`.

Run the tests with:

```sh
python3 -m unittest -v
```
